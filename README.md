## 📖 About This Guide

This is a comprehensive reference guide covering essential JavaScript concepts for backend developers working with Node.js. Rather than beginner basics, this guide dives into the mechanics that affect real-world production code.

### What You'll Find

1. **Event Loop & Concurrency Model** — Understand the execution order, microtasks vs macrotasks, and Node.js event loop phases
2. **Closures & Scope** — Lexical scoping, closure patterns, and common memory pitfalls
3. **Async Patterns** — Promises, async/await, error handling, and concurrency strategies
4. **Prototypes & Classes** — Prototype chain, inheritance, and modern class syntax
5. **Memory & Performance** — Garbage collection, memory leaks, and optimization techniques
6. **Modern ES Features** — ES2015+ syntax and features you'll encounter in production code
7. **Gotchas & Edge Cases** — Common traps and surprising JavaScript behaviors
